Predicting Maize Crop Yields Using Machine Learning

Project Overview

This project involves using machine learning algorithms to predict maize crop yields in Nyeri, Kenya. The goal was to leverage historical weather data and agricultural practices to forecast future crop performance and help farmers make more informed decisions.

Key features of the project:

  • Data Collection: We gathered historical data on weather patterns (temperature, rainfall, wind speed) and crop yields over the last decade.
  • Data Preprocessing: Cleaned and normalized data to handle missing values and outliers, making it suitable for machine learning models.
  • Modeling: We used multiple regression techniques (Linear Regression, Random Forest, etc.) and evaluated model performance using metrics like RMSE (Root Mean Square Error) and RÂ².
  • Results: The Random Forest model provided the best results, with a prediction accuracy of over 85%, allowing us to identify key factors influencing yield variations.

By applying this model, farmers can anticipate better or worse harvests, adjust their farming practices, and prepare for climate changes.

Technologies Used

  • Python
  • Machine Learning (Scikit-learn)
  • Pandas and NumPy (for data manipulation)
  • Matplotlib and Seaborn (for data visualization)
